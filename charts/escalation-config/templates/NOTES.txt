Breakglass escalation-config chart

This chart renders the following CRs using values from `values.yaml`:
- ClusterConfig - Cluster connection configuration
- BreakglassEscalation - Privilege escalation policies
- DebugSessionClusterBinding - Debug session template bindings

To render templates locally:

  helm template escalation-config ./charts/escalation-config

To install to a cluster:

  helm install escalation-config ./charts/escalation-config --namespace breakglass --create-namespace
